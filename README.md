## How to run
Please use Python 3.6 on Linux OS, make sure you installed the following modules before running:

- numpy
- matplotlib.pyplot
- tensorflow 1.15.2 (higher version dose NOT work!)

Then, just type **python3 main.py** in terminal to run.
